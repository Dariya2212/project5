# Новый виджет

## Описание:

Новый виджет - это обновление для личного кабинета клиента банка. Это виджет, который показывает несколько последних успешных банковских операций клиента.

## Установка:

1. Клонируйте репозиторий:
```
git clone https://github.com/Dariya2212/project5.git
```
2. Установите зависимости:
```
poetry install
```

## Тестирование

1. Необходимо установить зависимости:
```
poetry add --group dev pytest-cov
```
2. Запуск тестов:
```
pytest
```
3. Запуск тестов с оценкой покрытия:
```
poetry run pytest --cov
```
4. Дополнительно создан HTML файл для просмотра отчета о покрытии в папке "htmlcov"
